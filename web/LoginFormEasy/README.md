### Информация

```

```

### Описание

Сайт представляет собой логин форму, которая отправляет post запрос в формате json на базу данных mongodb. Участник должен прорвести nosql инъекцию в базу данных. (порт 1339)

### Запуск

```sh
cd deploy
docker-compose up -d --build 
```
